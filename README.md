# Stockpile Inventory Management System

A multi-company inventory management platform built with Django and Django REST Framework.

## Features

- Warehouse Management
- Product Tracking
- Stock In / Stock Out Transactions
- Low Stock Alerts
- Expiry Monitoring
- JWT Authentication
- Company Data Isolation
- PostgreSQL Integration

## Tech Stack

- Python
- Django
- Django REST Framework
- PostgreSQL
- HTML/CSS
- JavaScript

## Project Status

Currently under active development.
